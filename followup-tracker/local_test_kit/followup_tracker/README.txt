==============================================================
 FOLLOW-UP TRACKER  —  LOCAL PC ONLY
 Dr. Manoj Agarwal Clinic, Bareilly
==============================================================

This program runs ONLY on the clinic PC. There is NO website, NO server,
and NO hosting. It opens in your browser at a local address
(http://127.0.0.1:5000) that only this PC can see. All data stays on the PC.

FIRST TIME ON A PC:
   Double-click   setup_tracker.bat     (installs everything, once)

EVERY DAY:
   Double-click   "Open Followup Tracker"  shortcut on the Desktop
   -> the tracker starts and your browser opens automatically
   -> pick the two Docterz CSVs, click Process
   -> the Excel saves to the Output folder
   -> close the small black window when done

FULL INSTRUCTIONS:   see  DEPLOYMENT_GUIDE.txt
STAFF QUICK STEPS:    see  STAFF_GUIDE.txt

IMPORTANT: never delete the  data/  folder — it holds all patient
history and pending follow-ups.
