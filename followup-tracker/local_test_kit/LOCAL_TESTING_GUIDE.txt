GOAL OF THIS PHASE
------------------
Run the tracker on your own PC for a few days. It will point out
the actionables — missed follow-ups, probable dropouts, no-shows.
No WhatsApp yet. Once you are confident it works, we host it on the
VPS and then add WABA messaging.

WHAT THIS KIT CONTAINS
----------------------
  1_INSTALL_ONCE.bat              -> run one time to set up
  2_START_TRACKER.bat             -> run each time you use it
  EMPTY_followup_for_backfill.csv -> used only for backfill days
  followup_tracker\               -> the application
  LOCAL_TESTING_GUIDE.txt         -> this file

You will also need (from Docterz / your records):
  - DOCTERZ_PATIENT_LIST_UPTO_31_MAY_2026.csv   (the master list)
  - Consultation reports for 1, 2, 3, 4 June      (backfill)
  - Tonight: today's consultation report + tomorrow's follow-up list


============================================================
  PART A — ONE-TIME SETUP (do this first, once)
============================================================

STEP 1 — Install Python (skip if already installed)
   - Go to https://www.python.org/downloads/
   - Download the Windows installer, RUN it
   - IMPORTANT: on the first screen TICK "Add Python to PATH"
   - Click Install, then Finish

STEP 2 — Put this kit somewhere simple
   - Extract this whole kit to an easy location, e.g.:
       C:\followup_tracker_kit\
   - You should see the .bat files and the followup_tracker folder
     inside it.

STEP 3 — Run the installer
   - Double-click  1_INSTALL_ONCE.bat
   - A black window opens and installs the required packages
     (needs internet, takes 1-2 minutes)
   - When it says "Setup complete", press a key to close.

Done. You never need to run the installer again on this PC.


============================================================
  PART B — START THE TRACKER
============================================================

   - Double-click  2_START_TRACKER.bat
   - A black window opens and stays open  (this is the engine —
     leave it open while working)
   - Your browser opens automatically to  http://localhost:5000
   - Login password:  clinic2026
   - When done for the day: close the browser tab, then close the
     black window.


============================================================
  PART C — FIRST-TIME DATA LOAD (do once, in order)
============================================================

  ----- C1. Load the master list (patients up to 31 May) -----
   - In the tracker page, find the "First-Time Setup /
     Load Master" section.
   - Upload:  DOCTERZ_PATIENT_LIST_UPTO_31_MAY_2026.csv
   - Wait for the green success message
     (it will say how many patients were loaded — about 7,200+).
   - This builds your contacts master. Do this ONLY ONCE.

  ----- C2. Backfill 1-4 June (pull in new patients) -----
   For EACH day (1 Jun, then 2 Jun, 3 Jun, 4 Jun), do one run:
   - In the main "Process Daily Files" section:
       Consultation report = that day's Docterz consultation CSV
       Follow-up list      = EMPTY_followup_for_backfill.csv
   - Click Process.
   - Repeat for each of the four days, in date order.
   This adds any NEW patients seen on those days into your master,
   and records their visits. (We use the empty follow-up file
   because you are only adding patients, not follow-up reminders,
   for these catch-up days.)

   After C1 + C2, your master is current up to 4 June.


============================================================
  PART D — DAILY RUN (your routine from tonight onward)
============================================================

  Tonight (first real run):
   - Consultation report = TODAY'S consultation CSV (5 June)
   - Follow-up list      = TOMORROW'S follow-up CSV (6 June)
   - Click Process.

  Every day after:
   1. In Docterz, export:
        a) today's consultation report
        b) tomorrow's follow-up list
   2. Start the tracker (2_START_TRACKER.bat)
   3. Upload both files -> Process
   4. The summary shows counts: Probable Dropouts,
      Actionable Missed Follow-Ups, Grace Period, Due Today, etc.
   5. Click to download the Excel:  Followup_Audit_YYYY-MM-DD.xlsx
   6. Open the Excel -> "Staff Action Today" tab. That is your
      call list: who to contact, sorted by urgency.


============================================================
  WHAT TO CHECK DURING TESTING
============================================================
  - Do the patient names/numbers resolve correctly?
  - Does "Staff Action Today" show sensible no-shows / dropouts?
  - Are patients who returned correctly marked as Returned
    (not still shown as missed)?
  - Any names under "Identity Problems" that need a manual look?

  Run it daily for a few days. When the action list looks correct
  and useful, we move to VPS hosting, then add the WABA WhatsApp
  messages (post-call + follow-up reminders).
